Portable Local AI

Plug in. Unlock. Work privately.

Sentinel Drive is designed to make your AI environment portable without making it cloudy. The drive becomes the home for your SentinelAI configuration, encrypted vault, file workspace, browser sandbox, scanner state, local models, and automation memory.

The long-term product vision is simple: your AI operating system should move with you, but your data should stay physically and cryptographically under your control.

What It Carries

  • SentinelAI desktop runtime and worker layer
  • Encrypted vault for credentials, keys, notes, and secrets
  • Private browser profile that does not write to the host
  • Encrypted file storage and organization metadata
  • Local model suite and hardware-aware model manager
  • Guardian scanner history and quarantine state
  • Personal memory and Obsidian-compatible notes

Fingerprint/PIN Gate

Biometric-first unlock flow with PIN fallback planned for the physical device.

Zero Cloud Storage

Vault, files, browser state, and local metadata live on the drive, not in a hosted account.

Private Browser

Sandboxed browser session with vault-powered autofill and no persistent host writes.

Encrypted Storage

A portable encrypted workspace for files, tags, notes, and AI-assisted organization.

Guardian Scanner

Passive and active scanning layer for files added to the drive and suspicious host activity.

Local Models

Hardware-aware model suite so Sentinel can route tasks to the right local model.
